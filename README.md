# 📱 Android Developer en Evolución | Kotlin

Proyecto práctico de Android (nivel intermedio) desarrollado con Kotlin, enfocado en crecimiento
profesional y preparación para entornos reales de producto.

Este repositorio forma parte de mi evolución como **Android Developer Junior**, siguiendo mi Plan de
Desarrollo Individual (PDI) con el objetivo de adquirir mayor autonomía técnica, fortalecer mi
criterio de arquitectura y prepararme para desarrollar productos digitales escalables, especialmente
en el sector fintech.

---

## 🚀 Sobre este proyecto

En este proyecto aplico:

- Arquitectura MVVM
- Separación de responsabilidades
- Uso de ViewModel
- Organización orientada a escalabilidad
- Buenas prácticas de código

El objetivo no es solo que funcione, sino entender el **por qué** detrás de cada decisión técnica.

---

## 🎯 Objetivos Profesionales

- Consolidar fundamentos intermedios y avanzados de Android.
- Desarrollar mayor autonomía en la toma de decisiones técnicas.
- Mejorar mantenibilidad y escalabilidad del código.
- Simular dinámicas de desarrollo similares a un entorno empresarial.
- Evolucionar hacia desarrollo de productos fintech con arquitectura sólida.

---

## 🛠️ Tecnologías y conceptos aplicados
